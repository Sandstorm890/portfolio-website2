Recipe Book

Single page web-app built with React/Redux utilizing a Rails backend. Allows users to upload and view custom recipes in a streamlined and easy to understand UI. This project brought together elements from all of my other apps, and was incredibly satisfying to build from the ground up.

Ghost Tracker

SPA JavaScript frontend built on a Rails backend. A fun app that allows users to create records of 'ghosts' and view the records on a seamless, single-page interface. This was my first experience developing a web-app that doesn't require a page refresh, and really demonstrates how JS fundamentally changed the internet!

Hustle

I'm rather proud of this one! Utilizing the Ruby on Rails framework, this web-app enables users to sign up as a worker and browse and bid on job opportunities posted by employers. There is a lot of complex data handling that is going on under the hood here, and I loved the challenge of getting all of the components to work together.

Ghost Grabbers

Engages Sinatra framework to provide a simple and elegant user interface where users can sign up and create, modify, and delete information for others to view. Another fun app that helped me fully understand how technology can empower people to harness complex sets of data.

CLI Recipe App

Command line application that allows users to view detailed recipes returned by a JSON API through a variety of search methods. This was my first complete project I ever coded, and seeing my API calls in action for the first time was a really fun experience!

Nutrition App

Application I am currently building to assist users in accessing nutritional data as quickly and effortlessly as possible. Utilizes React/Redux to access the USDA's database of nutritional data via their API to display nutritional data on a wide variety of foods. Future plans include being able to 'favorite' ingredients and foods and specify information displayed to help efficiently navigate individual dietary needs.
